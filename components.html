What is LaunchKit?

LaunchKit is a free HTML landing page template specifically designed for developer tools and open source projects. It provides a ready-to-deploy solution with customizable components.

How do I customize the colors?

You can customize colors by updating CSS variables in the index.css file, or by switching between light and dark themes using the theme-dark class.

Is it mobile responsive?

Yes! LaunchKit is fully responsive and includes dedicated CSS files for tablet and mobile breakpoints to ensure optimal viewing on all devices.
